The Silver Server supports the following types:

(ASCII and EDCBIC are only for use of textfiles 
please use Image for any other type of media transfer)

ASCII (by default):
	Denoted as: A
	Command sequence = TYPE<space>A<CRLF>

EDCBIC (by choice):
	Denoted as: E
	Command sequence = TYPE<space>E<CRLF>

Image/Binary (by choice):
	Denoted as: I
	Command sequence = TYPE<space>I<CRLF>
